## Smart-Watch-Website-by-using-HTML-and-CSS
A modern landing page for a Smart Watch brand built using "HTML &amp; CSS" .   It showcases smartwatch features with a stylish UI.

## Overview
This project is designed to demonstrate an elegant landing page for a smart watch brand.
It combines minimal design with effective use of color, layout, and typography — ideal for portfolios and front-end design showcases.

## Features
* Gradient background with modern UI styling
* Navigation bar with hover effects
* Hero section with image and call-to-action button
* Feature cards for GPS tracking, heart rate, and calorie counter
* Fully responsive design for all screen sizes
